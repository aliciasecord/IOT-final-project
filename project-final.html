Final Prototype

Voice-Assisted Emergency Room Finder

In this final prototype, shown left, the Google Assistant guides the user to nearby hospitals and gives estimated wait times. However, when using Google's inline editor, I was unable to perform the API calls necessary to give real-time responses to user interactions. The information returned is all the same. Therefore, a web solution, shown below, demonstrates how those API calls will return data to user requests.


In this web solution, users can search for a specific hospital, find a nearby hospital via geolocation or address input, or see the full list of emergency rooms in the United States.

Next steps will involve figuring out the API calls and migrating all of the user interaction into the voice interaction. Further down the road, it would be interesting to look into the other metrics available from Data.Medicare.gov and supply other hospital metrics to users encountering specific medical emergencies.
