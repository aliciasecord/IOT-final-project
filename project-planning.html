Project Planning

Voice-Assisted Emergency Room Finder

There are two parts to this project: Accessing the necessary APIs and building the action for the Google Home.


After talking to Lena Groeger at ProPublica, I know their (more extensive) program used six datasets from the Center for Medicare and Medicaid Services and real-time traffic data from Google Maps. The first step here will be to learn how to access each API and filter for the relevant data. Does the data need to interact in any way? Do I need any access keys? Do I need all six data sets?


As for the Google Home action, I will build the voice interface with Google's Dialogflow platform. I haven't yet figured out how to integrate Javascript with Dialogflow, so that will be part of this discovery process.
